Erik's 20% Project: Programing

This it the site where I'll post all of my projects and games. I chose this project because I previously had some knowledge of programing and I wanted to learn more.

About:

The language I've chosen is Javascript. Before I started this project, I knew a little bit of a language called Python. Python, like Javascript, is object-orented. This means things can have properties instead of having to make separate variables for everything. I decided on Javascript because it's a lot like Python but you can have graphics.

For programming, I mainly use three APIs. They are Processing 3, Barckets, and P5.js. All the programs that you download were made in Processing. The web-based ones were built in P5.js using the Atom interface. Barckets is for web development and is what I used to make this website.
